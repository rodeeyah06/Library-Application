# Library Management System

A Java-based library management application that allows users to manage books, students, and teachers in a library system.

## Features

- Add and manage books in the library inventory
- Register students and teachers as library users
- Track book lending and returns
- User role management (Student, Teacher)
- Priority-based library access

## Technologies

- Java
- Object-oriented programming principles
- File-based data management

## Project Structure

The application includes the following main components:
- **Book**: Manages book information (title, author, copies)
- **User**: Base class for library users
- **Student**: Student user with level-based priority
- **Teacher**: Teacher user with library privileges
- **UserRole**: Defines different user roles in the system

## How to Run

1. Clone this repository
2. Navigate to the Library folder
3. Compile the Java files:
```
   javac org/example/model/*.java
```
4. Run the main application class
